# Generador Seguro de Contraseñas

## Descripción

Este proyecto consiste en el desarrollo de un generador seguro de contraseñas utilizando Python. El programa permite registrar un usuario y elegir si desea generar una contraseña automática o ingresar una contraseña manual.

## Objetivo

Desarrollar un programa funcional aplicando estructuras básicas de programación, como variables, condicionales, bucles y funciones. También se utiliza GitHub para gestionar el proyecto y RAPTOR para representar la lógica mediante diagramas de flujo.

## Funcionalidades

- Registro de nombre de usuario.
- Generación automática de contraseñas.
- Ingreso manual de contraseña.
- Validación de longitud mínima.
- Uso de estructuras condicionales.
- Uso de estructuras repetitivas.
- Código organizado mediante funciones.
- Comentarios en partes importantes del programa.

## Herramientas utilizadas

- Lenguaje de programación: Python.
- IDE: Visual Studio Code.
- Diagramas de flujo: RAPTOR.
- Repositorio: GitHub.

## Diagramas de flujo

Se elaboraron tres diagramas de flujo:

1. Diagrama general del programa.
2. Diagrama de generación automática de contraseña.
3. Diagrama de registro manual de contraseña.

Estos diagramas representan la lógica principal del sistema y se relacionan directamente con el código desarrollado.

## Estado del proyecto

El proyecto cuenta con un avance funcional. El programa permite registrar usuarios, generar contraseñas automáticas, ingresar contraseñas manuales y validar la longitud de la contraseña.
